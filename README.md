# 🗺️ U-Locate Blog de Ingeniería

Este es el blog de documentación técnica del proyecto **U-Locate**, desarrollado como proyecto final para las materias de **Sistemas de Información I** y **Multimedia** en la **Universidad Nacional Experimental del Táchira (UNET)**.

El blog detalla las fases de ingeniería de software aplicadas para resolver problemas de localización compleja en infraestructuras universitarias.

## 🚀 Tecnologías Utilizadas

- **Framework:** [Astro.build](https://astro.build/)
- **UI:** React & Tailwind CSS
- **Animaciones:** Framer Motion
- **Diseño:** Basado en la estética de Google Developers

## 📋 Fases del Proyecto

1. **Fase 1: Investigación de Hechos** - Aplicación de DCA y levantamiento de requerimientos basado en la experiencia estudiantil.
2. **Fase 2: Modelización** - Diseño del Diagrama Entidad-Relación (DER), Diagramas de Flujo de Datos (DFD) y Arquitectura de Red.
3. **Fase 3: Glosario de Proyecto** - Estandarización de la semántica técnica del sistema.
4. **Fase 4: Estrategia RAD** - Ciclos de desarrollo rápido e iteraciones del prototipo.

## 🛠️ Cómo ejecutar el proyecto localmente

1. Clonar el repositorio:
   ```bash
   git clone <url-del-repositorio>
   ```

2. Instalar las dependencias:
   ```bash
   npm install
   ```

3. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

4. Abrir [http://localhost:4321](http://localhost:4321) en tu navegador.
